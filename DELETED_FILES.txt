DELETED LEGACY FILES DOCUMENTATION

This file documents the legacy files that were removed from the project as they were not used in the production web application and did not affect the core logic.

- dashboard.html

- migrate_specs.js: Script to migrate product specifications

- replenish_stock.js: Script to replenish stock levels
- seed_mock_data.js: Script to seed mock data into database
- specs_backup.json: JSON backup of product specifications
- test_export.js: Script to test data export functionality
